Elixir
  • Built on top of Erlang
  • Concurrency & Parrallelism
  • Fast

Elixir is a compile functional language the runs on top of another language, Erlang. It compiles down to BEAM files. It is a great language for concurrency, parallelism and managing complicated process trees.

Elixir shares much of it's syntax with ruby. 

              def hello(name) do
                "Hello #{name}"
              end

It also has great error handling because it follows the philosophy of letting process fail. If a process fails, the supervisor of that process will restart it with a safe initial state. 

            # List utilities
            my_list = [1, 2, 3]

            [head | tail] = my_list

            IO.inspect(head) # => 1
            IO.inspect(tail) # => [2, 3]
